# Fofis
Uma biblioteca construída em C para processamento, leitura e manipulação
de dados em formato CSV.

## Build and Run
```bash
./build.sh
./main-dev
```

## Partes da aplicação
- [ ] Leitura do arquivo
- [ ] Escrita do arquivo
- [ ] Leitura de celula
- [ ] Alteração de celula
- [ ] Indexação
- [ ] Engine
- [ ] Tratamento de erro
- [ ] Tratamento de dados
